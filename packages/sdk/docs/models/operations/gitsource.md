# GitSource

Defines the Git Repository source to be deployed. This property can not be used in combination with `files`.

## Example Usage

```typescript
import { GitSource } from "@vercel/sdk/models/operations";

let value: GitSource = {
  owner: "<value>",
  ref: "<value>",
  slug: "<value>",
  type: "bitbucket",
};
```
